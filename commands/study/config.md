# 配置学习偏好

## 任务
引导学生了解配置选项，或应用他们提供的设置。在当前路径生成一份 `./study-config.md`文档作为学习偏好配置

## 配置选项

### 🎯 深度级别
- 小学（1-6年级）
- 初中（7-9年级）
- 高中（10-12年级）
- 本科
- 本科毕业阶段（学士学位）
- 硕士
- 博士候选人（Ph.D Candidate）
- 博士后
- 博士（Ph.D）
- [其他]

### 🧠 学习风格
- **视觉型**：偏好图片、示意图、图表
- **语言型**：偏好文字或口头解释
- **行动型**：通过实践和动手学习
- **直觉型**：倾向抽象概念与创新
- **反思型**：需要时间思考和分析
- **整体型**：先把握全局再关注细节
- [其他]

### 🗣️ 沟通风格
- **正式**：学术、专业的语言
- **教材式**：结构类似教科书
- **通俗**：日常、简单易懂的语言
- **叙事**：以故事方式呈现
- **苏格拉底式**：通过提问互动
- [其他]

### 🌟 语气风格
- **鼓励型**：支持且富有激励
- **中性**：客观、直截了当
- **信息型**：详尽且全面
- **友好型**：温暖、对话式
- **幽默型**：轻松、有趣
- [其他]

### 🔎 推理框架
- **演绎式**：从一般到具体
- **归纳式**：从具体到一般
- **溯因式**：寻找最佳解释
- **类比式**：通过比较进行学习
- **因果式**：强调因果关系
- [其他]

### 😀 表情符号
- **启用**：在描述中使用表情符号
- **禁用**：不使用表情符号

### 🌐 语言
- 英语
- 中文
- [其他]

### 📌 会话追踪（自动管理）
- `last_session_dir`: 记录最后活跃的学习会话
  - 由 `/review` 自动更新
  - 使 `/continue` 可以无参数直接继续学习
  - 无需用户手动配置

## 文档模板
````
**当前偏好：**
🎯 **学习深度**：[选择的级别]
🧠 **学习风格**：[选择的风格]
🗣️ **沟通风格**：[选择的风格]
🌟 **语气风格**：[选择的风格]
🔎 **推理框架**：[选择的框架]
😀 **表情符号**：[启用/禁用]
🌐 **语言**：[选择的语言]

```json
{
  "student_depth": "选择的级别",
  "learning_style": "选择的风格",
  "communication_style": "选择的风格",
  "tone_style": "选择的风格",
  "reasoning_framework": "选择的框架",
  "emoji_enabled": true,
  "language": "zh-CN",
  "last_session_dir": null
}
```

这些偏好将指导我今后的教学方式与互动方式。

💡 **下一步：**
- 发送 **/plan <主题>** 开始学习
- 想要调整任意设置时，再次使用 **/config**
````

## 重要提示
- 每次变更后务必展示当前配置 和 更变后的配置 (前后照应)
- 协助学生选择设置时保持支持性的态度
- 如学生提问，解释每个选项的含义
- 设置可随时更改
- 获得新配置后需立即适应
